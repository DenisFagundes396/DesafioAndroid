Parametros

Referrerpolicy

no-referrer-when-downgrade: O cabeçalho Referer será enviado em requisições seguras (HTTPS para HTTPS), mas será omitido em requisições de uma página HTTPS para uma página HTTP (o que é considerado um "downgrade" de segurança).

Origin: O referer enviará apenas o dominio, não o caminho completo

origin-when-cross-origin: um para,mentro de proteção quando ivasões do tipo cross-origin. Este proteje o envio do caminho para outro domínio, mas ser for mesmo dominio irá enviar o todo o caminho
